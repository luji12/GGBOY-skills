# 图像分析专家 Skill

## 核心工作流

处理图像分析任务时，按以下框架执行：

### 1. 画面要素提取（5W框架）
- **Who**: 画面中有哪些主体（人物/物体）？
- **What**: 主体在做什么？状态如何？
- **Where**: 场景环境是什么？光线条件？
- **When**: 时间氛围（白天/夜晚/特定时代）？
- **How**: 构图方式？景别？视角？

### 2. 视觉语言描述规范

#### 景别描述
- 远景（Extreme Wide Shot）：展示整体环境，人物极小
- 全景（Wide Shot）：人物完整可见，环境明显
- 中景（Medium Shot）：腰部以上，日常对话感
- 近景（Close-up）：面部/手部，情感表达
- 特写（Extreme Close-up）：局部细节，强调质感

#### 光影描述维度
- 光源方向：顺光/逆光/侧光/顶光/底光
- 光线质感：硬光（阴影清晰）/软光（柔和过渡）
- 色温：暖色（3000K）/中性（5500K）/冷色（7000K+）
- 影调：高调（浅淡）/低调（深暗）/中间调

#### 色彩体系描述
- 主色调 + 辅色 + 点缀色
- 饱和度：鲜艳/柔和/褪色/黑白
- 对比度：高对比/低对比/平衡

### 3. 构图原则
- **三分法**：主体置于黄金分割点
- **引导线**：线条引导视线至焦点
- **框架构图**：用前景元素框住主体
- **对称/非对称**：平衡感vs动态感
- **前后景深**：虚化分离主体与背景

### 4. 情感与氛围分析
- 色调与情绪对应：暖色=温暖/希望，冷色=疏离/压迫
- 空间感：逼仄=压迫，开阔=自由
- 动态感：静止=沉稳，模糊=紧张

### 5. 分镜单帧提示词生成规范

**格式模板**：
```
[视角/景别], [场景环境], [主体描述], [动作/状态], [情绪表达], [光线色调], [构图方式], [画面氛围], [风格参考]
```

**关键原则**：
- 只描述**这一刻**的静态画面，不描述动作过程
- 具体而不抽象（"眼眶泛红"而非"难过"）
- 光影和色调是情绪的语言
- 每个细节都服务于整体情绪
